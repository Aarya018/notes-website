üîπ 1. HTML Headings

HTML provides six levels of headings, from <h1> to <h6>.

  • <h1> is the largest and most important heading.
  • <h6> is the smallest and least important heading.

<h1>Main Heading</h1>
<h2>Sub Heading</h2>
<h3>Section Heading</h3>
<h4>Sub-section Heading</h4>
<h5>Minor Heading</h5>
<h6>Tiny Heading</h6>

üîπ 2. HTML Paragraphs

The <p> tag is used to define paragraphs in HTML. It automatically adds some spacing before and after the text.

  • Browsers add margin around paragraphs by default.
  • You can add multiple paragraphs in a page.
  • Line breaks within a paragraph can be added using <br>.

<p>This is a paragraph of text.</p>
<p>This is another paragraph.</p>

<p>This is a paragraph with a line break.<br>Here is the next line.</p>

üîπ 3. Anchor Tags

The <a> tag is used to define hyperlinks. It allows users to navigate to another page or a specific section on the same page.

  • href attribute defines the link destination.
  • target="_blank" opens the link in a new tab.
  • You can link to external sites or internal files.

<a href="https://www.google.com">Visit Google</a>

<a href="notes.html">Go to Notes Page</a>

<a href="#top" target="_blank">Open Top Section in New Tab</a>

üîπ 4. Images

The <img> tag is used to embed images in an HTML page. It's a self-closing tag and uses the following key attributes:

  • src: Specifies the path to the image file (URL or local path).
  • alt: Provides alternative text if the image cannot be displayed.
  • width and height: Used to control the size of the image.

<img src="images/logo.png" alt="Website Logo" width="200">

<img src="https://via.placeholder.com/150" alt="Sample Image">

üì® HTML Forms

Forms are used to collect user input. It can contain fields like text boxes, radio buttons, checkboxes, etc.


<form action="submit.php" method="post">
  <label>Name:</label>
  <input type="text" name="name" />

  <br />

  <label>Email:</label>
  <input type="email" name="email" />

  <br />

  <label>Gender:</label>
  <input type="radio" name="gender" value="male" /> Male
  <input type="radio" name="gender" value="female" /> Female

  <br />

  <label>Choose your country:</label>
  <select name="country">
    <option value="india">India</option>
    <option value="usa">USA</option>
  </select>

  <br />

  <label>Message:</label>
  <textarea name="message"></textarea>

  <br />

  <input type="submit" value="Submit" />
</form>
  • <form> tag wraps the form inputs.
  • action defines where to send form data.
  • method can be get or post.
  • Common input types: text, email, radio, checkbox, submit.
  • <select> is used for dropdowns.
  • <textarea> allows multi-line text input.

üîπ 5. HTML Lists

HTML provides three types of lists: Ordered, Unordered, and Description lists.

  • Unordered List: Uses <ul> and displays bullet points.
  • Ordered List: Uses <ol> and displays numbered items.
  • Description List: Uses <dl>, <dt>, and <dd> for terms and descriptions.

<!-- Unordered List -->
<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>

<!-- Ordered List -->
<ol>
  <li>Download Editor</li>
  <li>Write Code</li>
  <li>Run in Browser</li>
</ol>

<!-- Description List -->
<dl>
  <dt>HTML</dt>
  <dd>HyperText Markup Language</dd>
  <dt>CSS</dt>
  <dd>Cascading Style Sheets</dd>
</dl>

üîπ 6. HTML Tables

The <table> tag is used to create tables in HTML. Tables organize data into rows and columns.

  • <tr> defines a table row.
  • <td> defines a table cell (data cell).
  • <th> defines a header cell (bold and centered).
  • <thead>, <tbody>, and <tfoot> group sections of the table.

<table border="1">
  <thead>
    <tr>
      <th>Name</th>
      <th>Age</th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>Aarya</td>
      <td>18</td>
    </tr>
    <tr>
      <td>Raj</td>
      <td>20</td>
    </tr>
  </tbody>
</table>

üîπ 7. Semantic Tags

Semantic tags clearly describe their meaning in a human- and machine-readable way. They help improve the structure, readability, and SEO of your web pages.

  • <header>: Represents the top section of a page (often contains logo, nav).
  • <nav>: Defines navigation links.
  • <main>: Contains the main content of the document.
  • <section>: Groups related content.
  • <article>: Represents independent, self-contained content.
  • <aside>: Defines content aside from the main content (like a sidebar).
  • <footer>: Represents the footer of the page.

<header>
  <h1>Welcome to My Website</h1>
</header>

<nav>
  <a href="#home">Home</a>
  <a href="#about">About</a>
</nav>

<main>
  <section>
    <article>
      <h2>Blog Post</h2>
      <p>This is a blog post.</p>
    </article>
  </section>
</main>

<aside>This is a sidebar.</aside>

<footer>
  <p>¬© 2025 Aarya</p>
</footer>

üîπ 8. Inline vs Block Elements

Block-level elements start on a new line and take up the full width, while inline elements stay in the same line and only take as much width as needed.

  • Block Elements: <div>, <p>, <h1> to <h6>, <section>, <article>
  • Inline Elements: <span>, <a>, <strong>, <em>, <img>

<!-- Block Example -->
<div>
  <p>This is a block element</p>
</div>

<!-- Inline Example -->
<p>
  This is an <span>inline</span> element inside a paragraph.
</p>

üîπ 9. HTML Entities

HTML entities are used to display reserved or special characters that cannot be typed directly in HTML.

  • Entities start with & and end with ;.
  • They are helpful for displaying characters like <, >, &, etc.
  • You can also use them to display non-keyboard characters like copyright & currency symbols.

<p>Less than: &lt;</p>
<p>Greater than: &gt;</p>
<p>Ampersand: &amp;</p>
<p>Copyright: &copy;</p>
<p>Rupee Symbol: ‚Çπ (₹) or &#8377;</p>

üîπ 10. HTML Comments

HTML comments are used to leave notes or explanations within the HTML code. They do not appear on the web page when viewed in a browser.

  • They are helpful for documentation and making your code easier to understand.
  • Browsers ignore the content inside comments.
  • Syntax: <!-- This is a comment -->

<!-- This is a comment -->

<p>This paragraph is visible.</p>

<!-- <p>This paragraph is hidden in the browser.</p> -->

üîπ 11. HTML Media Tags

HTML provides built-in support for embedding media like audio and video using the <audio> and <video> tags.

  • <audio> is used to embed sound content.
  • <video> is used to embed video content.
  • Use the controls attribute to display playback controls.
  • You can use source tags inside for multiple formats.

<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
  Your browser does not support the audio tag.
</audio>

<br>

<video width="320" height="240" controls>
  <source src="video.mp4" type="video/mp4">
  Your browser does not support the video tag.
</video>

üîπ 12. HTML Iframe

The <iframe> (Inline Frame) tag is used to embed another HTML page or external content (like a YouTube video or map) within your current page.

  • src: Specifies the URL of the page to embed.
  • width and height: Control the size of the iframe.
  • frameborder: (Optional) Specifies the border (usually 0 or 1).
  • allowfullscreen: Enables fullscreen for embedded videos.

<!-- Embedding a website -->
<iframe src="https://example.com" width="400" height="300">
  Your browser does not support iframes.
</iframe>

<br>

<!-- Embedding a YouTube Video -->
<iframe width="560" height="315" 
  src="https://www.youtube.com/embed/dQw4w9WgXcQ" 
  frameborder="0" allowfullscreen>
</iframe>
