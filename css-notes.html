üéØ CSS Selectors

CSS selectors are used to "select" the HTML elements you want to style.

  • *: Selects all elements
  • p: Selects all <p> elements
  • .class: Selects elements with a specific class
  • #id: Selects an element with a specific ID
  • element, element: Group selector
  • element element: Descendant selector
  • element > element: Direct child selector

* {
  margin: 0;
}

p {
  color: red;
}

.box {
  border: 1px solid black;
}

#header {
  background-color: lightgray;
}

div > p {
  font-size: 18px;
}

üåà CSS Colors

CSS supports different formats for colors:

  • Color Name: red, blue
  • Hex Code: #ff0000
  • RGB: rgb(255, 0, 0)
  • RGBA: rgba(255, 0, 0, 0.5)
  • HSL: hsl(0, 100%, 50%)

body {
  background-color: #fffaf0;
  color: rgb(0, 0, 0);
}

h1 {
  color: hsl(210, 50%, 40%);
}

‚úçÔ∏è Text Styling

Use these properties to control the appearance of text:

  • font-family: Sets the font type
  • font-size: Controls text size
  • font-weight: Controls boldness
  • text-align: Aligns text (left, center, right)
  • text-decoration: Underline, overline, etc.
  • line-height: Sets line spacing
  • letter-spacing: Controls spacing between letters

h1 {
  font-family: 'Poppins', sans-serif;
  font-size: 2rem;
  font-weight: bold;
  text-align: center;
  text-decoration: underline;
}

üì¶ CSS Box Model

Every HTML element is a box with 4 parts:

  • Content: Actual content (text/image)
  • Padding: Space around content
  • Border: Edge around padding
  • Margin: Space outside border

div {
  padding: 20px;
  border: 2px solid brown;
  margin: 30px;
}

üñºÔ∏è CSS Display

The display property defines how an element is displayed on the page.

  • block: Starts on a new line (e.g., <div>)
  • inline: Stays in line (e.g., <span>)
  • inline-block: Inline but allows box properties
  • flex: Flexbox layout
  • grid: Grid layout
  • none: Hides the element

.container {
  display: flex;
  gap: 20px;
}

üìç CSS Position

The position property is used to set how an element is positioned in the document.

  • static: Default (normal flow)
  • relative: Positioned relative to its normal position
  • absolute: Positioned relative to the nearest positioned ancestor
  • fixed: Stays fixed while scrolling
  • sticky: Scrolls with the page but sticks when it reaches a position

.sticky-bar {
  position: sticky;
  top: 0;
  background: brown;
}

üì± Media Queries

Media queries make your site responsive by applying different styles for different screen sizes.

  • min-width: Apply styles above a certain width
  • max-width: Apply styles below a certain width
  • Common breakpoints:
    • Mobile: max-width: 600px
    • Tablet: max-width: 768px
    • Desktop: min-width: 1024px

@media (max-width: 768px) {
  .nav-links {
    flex-direction: column;
  }
}

üì¶ CSS Flexbox

Flexbox (Flexible Box Layout) is used to arrange elements in a row or column, making layout alignment easy and flexible.

  • display: flex or display: inline-flex enables Flexbox.
  • Main Axis: Direction defined by flex-direction (default is row).
  • Cross Axis: Perpendicular to main axis.
  • Important container properties:
    • flex-direction: row, row-reverse, column, column-reverse
    • justify-content: Aligns items horizontally (center, space-between, etc.)
    • align-items: Aligns items vertically (center, flex-start, stretch)
    • flex-wrap: Allows wrapping (wrap, nowrap)
    • gap: Adds space between flex items
  • Item properties:
    • flex: Shorthand for flex-grow, flex-shrink, and flex-basis
    • order: Controls item order
    • align-self: Overrides align-items for a single item

.container {
  display: flex;
  flex-direction: row;
  justify-content: space-between;
  align-items: center;
  flex-wrap: wrap;
  gap: 20px;
}

.item {
  flex: 1;
  order: 2;
  align-self: flex-end;
}

üî≤ CSS Grid

CSS Grid is a two-dimensional layout system used for building web page layouts efficiently.

  • Use display: grid to create a grid container.
  • Define columns and rows using grid-template-columns and grid-template-rows.
  • Use gap to add space between rows and columns.
  • Place items using properties like:
    • grid-column and grid-row
    • grid-area

.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-template-rows: auto;
  gap: 10px;
}

.item1 {
  grid-column: 1 / 3;
  grid-row: 1;
}

üîÑ CSS Transition

The transition property allows smooth animation between CSS property changes over a duration.

  • Use it to animate on hover, focus, or state change.
  • Common transition properties include:
    • transition-property
    • transition-duration
    • transition-timing-function
    • transition-delay
  • Use shorthand like transition: all 0.3s ease;

.box {
  background-color: red;
  transition: background-color 0.5s ease;
}

.box:hover {
  background-color: blue;
}

üéûÔ∏è CSS Animation

The animation property is used to animate HTML elements using @keyframes.

  • Use @keyframes to define the animation steps.
  • Main sub-properties:
    • animation-name
    • animation-duration
    • animation-timing-function
    • animation-delay
    • animation-iteration-count
    • animation-direction
  • Shorthand example: animation: bounce 2s ease-in-out infinite;

@keyframes bounce {
  0%   { transform: translateY(0); }
  50%  { transform: translateY(-100px); }
  100% { transform: translateY(0); }
}

.ball {
  animation: bounce 1s ease-in-out infinite;
}

üñºÔ∏è CSS Object Properties

CSS object-fit and object-position are used to control how an image or video is resized and positioned within its container.

üìå object-fit

Controls how the content (image/video) should be resized to fit its container.

  • fill: Stretches the content to fill the box (default).
  • contain: Scales the content to fit inside the box while maintaining aspect ratio.
  • cover: Scales the content to fill the box while maintaining aspect ratio. Some parts may be cropped.
  • none: Keeps original size.
  • scale-down: Same as none or contain, whichever is smaller.

img {
  width: 300px;
  height: 200px;
  object-fit: cover;
}

üìç object-position

Controls the alignment of the content inside the box.

  • Values like center, top, bottom, left, right
  • Can also use percentage or length units

img {
  object-position: top right;
}

üé® CSS Filter Property

The filter property is used to apply visual effects (like blur, brightness, grayscale) to elements such as images, text, or divs.

üß™ Common Filter Functions

  • blur(px): Applies a blur effect.
  • brightness(%): Adjusts brightness (100% is default).
  • contrast(%): Adjusts contrast.
  • grayscale(%): Converts to grayscale (100% is fully gray).
  • sepia(%): Adds a warm brown tone.
  • invert(%): Inverts the colors.
  • saturate(%): Increases or decreases color saturation.
  • hue-rotate(deg): Rotates hue angle (changes color).
  • drop-shadow(x y blur color): Adds a shadow similar to box-shadow but works inside filter.

img {
  filter: grayscale(100%) blur(2px) brightness(120%);
}
