üéØ CSS Selectors

CSS selectors are used to "select" the HTML elements you want to style.

  • *: Selects all elements
  • p: Selects all <p> elements
  • .class: Selects elements with a specific class
  • #id: Selects an element with a specific ID
  • element, element: Group selector
  • element element: Descendant selector
  • element > element: Direct child selector

* {
  margin: 0;
}

p {
  color: red;
}

.box {
  border: 1px solid black;
}

#header {
  background-color: lightgray;
}

div > p {
  font-size: 18px;
}

üåà CSS Colors

CSS supports different formats for colors:

  • Color Name: red, blue
  • Hex Code: #ff0000
  • RGB: rgb(255, 0, 0)
  • RGBA: rgba(255, 0, 0, 0.5)
  • HSL: hsl(0, 100%, 50%)

body {
  background-color: #fffaf0;
  color: rgb(0, 0, 0);
}

h1 {
  color: hsl(210, 50%, 40%);
}

‚úçÔ∏è Text Styling

Use these properties to control the appearance of text:

  • font-family: Sets the font type
  • font-size: Controls text size
  • font-weight: Controls boldness
  • text-align: Aligns text (left, center, right)
  • text-decoration: Underline, overline, etc.
  • line-height: Sets line spacing
  • letter-spacing: Controls spacing between letters

h1 {
  font-family: 'Poppins', sans-serif;
  font-size: 2rem;
  font-weight: bold;
  text-align: center;
  text-decoration: underline;
}

üì¶ CSS Box Model

Every HTML element is a box with 4 parts:

  • Content: Actual content (text/image)
  • Padding: Space around content
  • Border: Edge around padding
  • Margin: Space outside border

div {
  padding: 20px;
  border: 2px solid brown;
  margin: 30px;
}

üñºÔ∏è CSS Display

The display property defines how an element is displayed on the page.

  • block: Starts on a new line (e.g., <div>)
  • inline: Stays in line (e.g., <span>)
  • inline-block: Inline but allows box properties
  • flex: Flexbox layout
  • grid: Grid layout
  • none: Hides the element

.container {
  display: flex;
  gap: 20px;
}

üìç CSS Position

The position property is used to set how an element is positioned in the document.

  • static: Default (normal flow)
  • relative: Positioned relative to its normal position
  • absolute: Positioned relative to the nearest positioned ancestor
  • fixed: Stays fixed while scrolling
  • sticky: Scrolls with the page but sticks when it reaches a position

.sticky-bar {
  position: sticky;
  top: 0;
  background: brown;
}

üì± Media Queries

Media queries make your site responsive by applying different styles for different screen sizes.

  • min-width: Apply styles above a certain width
  • max-width: Apply styles below a certain width
  • Common breakpoints:
    • Mobile: max-width: 600px
    • Tablet: max-width: 768px
    • Desktop: min-width: 1024px

@media (max-width: 768px) {
  .nav-links {
    flex-direction: column;
  }
}

üì¶ CSS Flexbox

Flexbox (Flexible Box Layout) is used to arrange elements in a row or column, making layout alignment easy and flexible.

  • display: flex or display: inline-flex enables Flexbox.
  • Main Axis: Direction defined by flex-direction (default is row).
  • Cross Axis: Perpendicular to main axis.
  • Important container properties:
    • flex-direction: row, row-reverse, column, column-reverse
    • justify-content: Aligns items horizontally (center, space-between, etc.)
    • align-items: Aligns items vertically (center, flex-start, stretch)
    • flex-wrap: Allows wrapping (wrap, nowrap)
    • gap: Adds space between flex items
  • Item properties:
    • flex: Shorthand for flex-grow, flex-shrink, and flex-basis
    • order: Controls item order
    • align-self: Overrides align-items for a single item

.container {
  display: flex;
  flex-direction: row;
  justify-content: space-between;
  align-items: center;
  flex-wrap: wrap;
  gap: 20px;
}

.item {
  flex: 1;
  order: 2;
  align-self: flex-end;
}
