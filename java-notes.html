üì¶ JavaScript Variables

Variables are containers for storing data values. In JavaScript, you can use var, let, or const to declare variables. Each has different scoping and re-declaration rules.

üí° Rules for Naming Variables

  • Can include letters, digits, underscores (_), and dollar signs ($)
  • Must begin with a letter, _, or $
  • Cannot be a JavaScript reserved word
  • Case-sensitive (harry and HaRry are different)

üîç var, let & const

  • var ‚Üí Function-scoped or globally scoped
  • let ‚Üí Block-scoped, can be updated but not re-declared in same scope
  • const ‚Üí Block-scoped, must be initialized, cannot be updated or re-declared

// var example
var firstName = "Aarya";
var score1 = 50;
var score2 = 70;
console.log(firstName, score1 + score2); // Aarya 120

// let example
let age = 18;
age = 19; // allowed
console.log(age); // 19

// const example
const pi = 3.1416;
// pi = 3.15; // ‚ùå Error
console.log(pi);

üìÇ JavaScript Data Types

Data types define the type of data that can be stored in a variable.

üîπ Primitive Data Types

  • string ‚Üí Text data
  • number ‚Üí Integer or floating-point numbers
  • boolean ‚Üí true/false values
  • undefined ‚Üí Variable declared but not assigned
  • null ‚Üí Intentional empty value
  • symbol ‚Üí Unique identifiers
  • bigint ‚Üí Large integers

üîπ Non-Primitive Data Types

Objects: Key-value pairs storing multiple values in a single variable.


// Primitive examples
let city = "Mumbai";
let temperature = 30.5;
let isSunny = true;
let notAssigned;
let emptyValue = null;
console.log(typeof city, typeof temperature, typeof isSunny, typeof notAssigned, typeof emptyValue);

// Object example
let car = {
  brand: "Tesla",
  model: "Model S",
  year: 2024
};
console.log(car);

// Adding new property
car.color = "Red";
console.log(car);

üîÄ JavaScript if...else Statement

The if...else statement is used to perform different actions based on conditions.

  • if ‚Üí Executes code if the condition is true.
  • else ‚Üí Executes code if the condition is false.
  • else if ‚Üí Checks another condition if the first one is false.

Syntax:


if (condition) {
  // code runs if condition is true
} else if (anotherCondition) {
  // code runs if anotherCondition is true
} else {
  // code runs if all conditions are false
}

Example:


let age = 18;

if (age >= 18) {
  console.log("You are an adult");
} else {
  console.log("You are a minor");
}

let marks = 75;

if (marks >= 90) {
  console.log("Grade: A+");
} else if (marks >= 75) {
  console.log("Grade: A");
} else if (marks >= 50) {
  console.log("Grade: B");
} else {
  console.log("Fail");
}
